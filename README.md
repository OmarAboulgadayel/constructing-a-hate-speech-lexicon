# Constructing a Hate Speech Lexicon
Note: This paper explores the topic of hate speech detection, and as a result, contains strong language and explicit references to offensive or discriminatory remarks. The purpose of using such language is solely for the purpose of accurately discussing and analyzing hate speech within the context of this research. Reader discretion is advised, and this paper is intended for academic purposes only.

## Repository
* `./data` contains the hate speech dataset provided in the paper "Automated Hate Speech Detection and the Problem of Offensive Language" by Thomas Davidson, Dana Warmsley, Michael Masey and Ingmar Weber.

* `./lexicon` contains the constructed hate speech lexicon.

* `./src` contains the Python program used to create the hate speech lexicon.
